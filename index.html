❓ Preguntas Frecuentes

💬 Dudas comunes sobre HTML
  • ¿Con qué programa edito archivos HTML?
    Usá Visual Studio Code o cualquier editor de texto. ¡Hasta el Bloc de notas sirve! 💻
  • ¿Cómo veo mi archivo HTML en el navegador?
    Guardá el archivo con extensión .html y abrilo con doble clic. O arrastralo al navegador.
  • ¿Para qué sirven las etiquetas <h1>, <p>, <ul>...?
    Para estructurar el contenido. Pensalas como cajas que ordenan el texto y los elementos de tu web.
  • ¿Por qué no se ve mi imagen?
    Revisá que el nombre y la ruta del archivo estén bien escritos. ¡Y no te olvides del alt! 🖼️
  • ¿Qué es eso de </html> o </body>?
    Son etiquetas de cierre. Todo lo que abrís con <etiqueta> se cierra con </etiqueta>.
